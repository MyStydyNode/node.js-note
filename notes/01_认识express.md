# 01_认识express

### 前言

- nodejs中文官网     http://nodejs.cn/

- express 官网          http://expressjs.com/

  express 中文官网  https://www.expressjs.com.cn/

### 知识点

- 初始化 package.json

  ```Script
  $ npm init
  ```

- 安装 express

  ```Script
  $ npm install express --save
  ```

- 新建 server.js,在里面写代码，用脚本执行

  ```Script
  $ node server.js
  ```

  注：不能操作dom元素，只能操作存粹的js
